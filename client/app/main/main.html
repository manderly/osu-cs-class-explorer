About this project

OSU CS Course Explorer was developed for students of Oregon State University's online Computer Science program. It uses data collected from real students on the OSUOnlineCS subreddit. The data in this app is as useful as students like you make it. Add your own course review(s) via the survey here.

App developed by Mandi Grant, OSU CS student and professional web developer. Explore the project code on my GitHub and connect with me on LinkedIn.
